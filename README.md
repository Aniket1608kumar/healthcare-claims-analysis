# 🏥 Healthcare Claims Data Analysis

This project focuses on analyzing simulated healthcare claims data using Python. It demonstrates how data analysis techniques can be applied to extract meaningful insights from raw data.

---

## 📌 Project Overview

The goal of this project is to explore and analyze healthcare claim records to understand patterns in claim distribution, approval rates, and financial trends across departments.

---

## 🛠️ Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## 📊 Dataset

This project uses a **simulated healthcare claims dataset** created using Python and NumPy.

### Features include:

- **Department** → Department handling the claim (IT, HR, Sales, etc.)
- **Claim Amount** → Amount requested in the claim
- **Approval Status** → Whether the claim was approved (1) or rejected (0)

---

## 🔍 Key Analysis Performed

- 📈 Department-wise claim distribution
- 💰 Average claim amount by department
- ✅ Claim approval rate analysis
- 📊 Visualization of claim amount distribution
