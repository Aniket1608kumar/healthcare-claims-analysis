# Healthcare Claims Data Analysis

This project analyzes simulated healthcare claims data using Python.

## Tools Used
- Python
- Pandas
- Matplotlib

## Dataset
This project uses a simulated healthcare claims dataset containing:

- Department
- Claim Amount
- Approval Status

The dataset was generated using Python and NumPy for demonstration purposes.

## Key Analysis
- Department claim distribution
- Average claim amount
- Approval rate
- Claim amount distribution

## Purpose
This project demonstrates basic data analysis using Python, Pandas, and data visualization techniques.
